## ⚠️ 注意事项

- **Windows 端安装**：安装时请避免使用中文路径，建议使用英文路径，以免出现兼容性问题
- **视频源配置**：建议优先使用在线链接配置视频源，本地包在部分设备上可能存在识别问题，iOS 设备最为明显
- **JAR包支持**：安卓端正式支持JAR包运行，配置方式与影视源相同
- **小说功能**：需要在接口配置中添加"小说"类型源才会显示小说Tab
- **道长项目使用**：
  - 使用drpy2时，如遇到部分网页源兼容性问题，请及时反馈
  - 启动了drpys后，如需切换另一个node接口，必须先退出应用并清理后台，再进行切换
  - iOS暂不支持道长项目相关功能
- **iOS用户特别注意**：如需切换nodejs接口，请务必按照v1.0.1中的步骤操作，避免应用闪退
- **TMDB功能**：需要自己申请TMDB API Key才能使用媒体库功能
- **Win端优化**：大概率修复了win端node进程残留问题

## 项目简介

**PeekPili** 是基于 [PiliPlus](https://github.com/bggRGjQaUbCoE/PiliPlus) 进行二次开发的应用，在保留原版 PiliPlus 全部功能的基础上，新增了 **TVbox 视频源播放功能**，支持多种视频源格式，让您可以在一个应用中享受丰富的第三方视频源内容。

> 📌 **关于 PiliPlus 原版功能**  
> 本项目完整保留了 PiliPlus 的所有功能，如需了解原版的详细功能特性，请访问原版 GitHub 仓库：  
> **[👉 PiliPlus 原版仓库](https://github.com/bggRGjQaUbCoE/PiliPlus)**

**✨ 多平台支持** - PeekPili 支持 Android、iOS、Pad、Windows 等平台，无论您使用什么设备，都能享受一致的使用体验。

- **🐧 企鹅群：756526268**

## 核心功能

### 📚 小说阅读模块（v1.0.6新增）

**PeekPili** 现已支持完整的小说阅读功能，为您提供沉浸式的阅读体验。

#### 🎯 智能显示机制
- **动态Tab显示**：当接口配置中存在"小说"类型源时，自动在底部导航栏显示小说Tab
- **多种识别方式**：支持 `"类型": "小说"`、名称包含"小说"、key包含"novel"/"book"等识别方式
- **多书源支持**：可在不同书源间自由切换，享受丰富的阅读资源

#### 📖 强大的阅读器功能
- **个性化设置**：字体大小(14-32px)、行高(1.4-2.5倍)、字间距、段间距、边距等全面可调
- **丰富主题**：浅色主题7种、深色主题5种，多种背景纹理可选
- **多种翻页模式**：滚动、左右滑动、覆盖翻页、仿真翻页
- **高级功能**：TTS语音朗读、自动翻页、书签、笔记高亮、屏幕常亮
- **PC端支持**：完整的键盘快捷键支持，方向键、PageUp/PageDown等

#### 📚 完善的管理功能
- **书架管理**：收藏小说集中管理，阅读进度实时显示
- **阅读历史**：按日期智能分组，显示最后阅读章节
- **搜索功能**：关键词搜索，快速定位目标小说

> 📖 **详细说明**

### 📺 TVbox 视频源功能

本项目在 PiliPlus 基础上新增了完整的 TVbox 视频源播放功能，支持多种视频源格式。

#### 🚀 JAR包全面支持（v1.0.6新增）

- **安卓端JAR包支持** - 正式支持JAR包运行，基本适配所有JAR方法
- **GO代理播放** - 支持GO代理播放功能，提升播放体验
- **配置简单** - JAR包配置方式与影视源完全相同，导入即用
- **测试验证** - 目前已测试VOX和摸鱼JAR包

#### 🎬 震撼更新 - TMDB媒体库支持（v1.0.4优化）

- **完整的TMDB API服务** - 提供精美的媒体库界面体验
- **横竖屏适配** - 支持深色和浅色主题切换
- **沉浸式体验** - 让TVBox也能体验到Emby、Infuse等媒体库的极致体验
- **智能匹配** - 进入详情页前自动匹配大海报，匹配不准确时可自行编辑调整
- **智能缓存** - 匹配后直接加载，提升使用体验
- **详情增强** - 可查看分季信息、分集剧情以及演员详情
- **需要自己申请TMDB API Key**

#### 🎭 道长项目全平台适配（v1.0.3新增，v1.0.5持续优化）

- **全平台适配道长项目** - 支持道长drpys本地搭建（iOS暂不支持）
- **DS更新至1.3.15** - 支持道长离线版dr2
- **兼容hipy** - 兼容DS里面t3类型的hipy
- **实验室功能** - 新增道长项目本地搭建，具体操作说明见应用内介绍
- **设置中心交互优化** - 适配道长的动作交互，设置中心可以直接点击，不用跳浏览器
- **drpy2适配** - 适配drpy2，部分网页源可能有兼容问题
- **JS源本地代理** - 支持js源本地代理

#### 支持的视频源类型

本项目全平台支持以下三种类型的视频源：

1. **T4 源（TVbox Type4）**
   - 支持标准 T4 API 接口源
   - 支持 HTTP API 格式的视频源
   - 完整的视频分类、搜索、详情、播放功能
   - 支持多站源切换和管理
   - 支持解析接口配置

2. **T3 源 - CatJS 源（猫源）**
   - 支持 cat 格式的 JavaScript 爬虫源
   - 支持 ES Module 格式的 JS 源
   - 支持 **Open 版本**和 **NodeJS 版本**两种猫源格式
   - 内置常用工具库（CryptoJS、lodash、cheerio 等）
   - 支持远程 URL 和本地代码两种方式

3. **T3 源 - Python 源（py 源）**
   - 支持 Python 爬虫脚本源
   - 完整的爬虫接口支持

#### 源助手功能

为了方便用户测试和调试视频源，项目内置了强大的**源助手**工具：

- **CatJS 源调试助手**
  - 支持测试 cat 格式的 JS 爬虫源
  - 支持远程 URL 和代码输入两种模式
  - 可测试所有爬虫方法：
    - `homeContent` - 首页分类
    - `homeVideoContent` - 首页推荐视频
    - `categoryContent` - 分类内容
    - `searchContent` - 搜索内容
    - `detailContent` - 详情内容
    - `playerContent` - 播放内容
  - 实时查看测试结果，支持 JSON 格式化显示
  - 支持清除缓存，方便重新测试

- **Python 源测试助手**
  - 支持测试 Python 爬虫源
  - 支持 URL 和代码两种输入方式
  - 完整的错误提示和调试信息

## 使用说明

### 小说功能使用（v1.0.6新增）

1. **配置书源**
   - 在接口配置中添加 `"类型": "小说"` 字段
   ```json
   {
     "sites": [
       {
         "key": "novel_source",
         "name": "novel",
         "api": "https://example.com/api/novel",
         "type": 3,
         "类型": "小说"
       }
     ]
   }
   ```

2. **开始阅读**
   - 配置完成后，小说Tab会自动出现在底部导航栏
   - 点击小说Tab，选择书源，即可开始阅读

3. **阅读器使用**
   - 双击屏幕呼出/隐藏控制栏
   - 左滑上一页，右滑下一页
   - PC端支持键盘快捷键（方向键、PageUp/PageDown、空格键等）

4. **个性化设置**
   - 在阅读器中可调节字体、主题、翻页模式等
   - 支持TTS语音朗读、自动翻页等高级功能

> 📖 **详细说明**  
> 更多小说功能的详细使用方法，请参考：[小说模块功能文档](README_NOVEL.md)

### TVbox 功能使用

1. **配置视频源**
   - 进入设置页面，找到 TVbox 相关设置
   - 配置视频源接口地址（支持本地和远程配置）
   - 选择要使用的视频源
   - **JAR包配置**（v1.0.6新增）：JAR包配置方式与影视源完全相同，导入即用

2. **使用源助手**
   - 进入源助手页面
   - 选择要测试的源类型（CatJS 或 Python）
   - 输入源地址或代码
   - 选择测试方法并填写参数
   - 查看测试结果

3. **浏览和播放**
   - 在 TVbox 模式下浏览视频内容
   - 支持分类筛选和搜索
   - 支持站源指定搜索和详情页快搜（v1.0.5新增）
   - 选择播放线路和画质
   - 享受流畅的播放体验
   - **TMDB媒体库体验**（v1.0.4优化）
     - 配置TMDB API Key后，可享受精美的媒体库界面
     - 自动匹配大海报和详细信息
     - 查看分季信息、分集剧情和演员详情

4. **道长项目使用**（v1.0.3新增，v1.0.5持续优化）
   - 实验室中可进行道长项目本地搭建（iOS暂不支持）
   - 支持drpy2格式的网页源和道长离线版dr2
   - 设置中心可直接点击操作，无需跳转浏览器
   - 兼容DS里面t3类型的hipy

5. **高级功能使用**（v1.0.6新增）
   - **WebDAV功能**：支持通过WebDAV协议进行文件管理和同步

## 常见问题

### Q: 有没有 TV 端？

**A:** 没有。本项目目前不支持 TV 端。

> 💡 **推荐**  
> 如果您仅仅是安卓手机和 TV 用户，强烈推荐使用 **ok影视**，因为兼容性最强、覆盖性最广。

### Q: 支不支持 jar 包？

**A:** 安卓端正式支持JAR包运行（v1.0.6新增）。配置方式与影视源完全相同，基本适配所有JAR方法，支持GO代理播放功能。目前已测试VOX和摸鱼JAR包。

### Q: 播放是否支持本地代理（如 127.0.0.1:9978 格式）？

**A:** Python源和JAR包（v1.0.6新增）支持。JAR包支持GO代理播放功能。

### Q: 小说功能如何使用？

**A:** 需要在接口配置中添加 `"类型": "小说"` 字段，配置完成后小说Tab会自动显示。支持多种识别方式，包括名称包含"小说"、key包含"novel"/"book"等。详细使用说明请查看[小说模块功能文档](README_NOVEL.md)。

### Q: 小说阅读器支持哪些功能？

**A:** 支持个性化设置（字体、主题、翻页模式等）、TTS语音朗读、自动翻页、书签、笔记高亮、屏幕常亮等功能。PC端还支持完整的键盘快捷键操作。
